How to Use ICI Collaborative Memory Safely & Privately

What is ICI Collaborative Memory?

ICI is your AI-powered memory companion. It helps you remember, organize, and share important information in a way that is both personal and collaborative. You control what is private, what is shared, and with whom.

Memory Scopes

  • Private Memory: Only you can see and use this information. It is never shared unless you choose to.
  • IP Shared Memory: Shared with trusted devices on your local network (like your home or office).
  • Community Shared Memory: Shared with other ICI users you connect with. Great for group projects or family reminders.

How to Use ICI Safely

  1. Decide what to keep private. Only add sensitive information to your Private Memory.
  2. Review sharing settings. Before sharing, check who will have access to your memory or chat.
  3. Use QR code authentication to securely link your devices. Never share your QR code with people you don't trust.
  4. Ask for confirmation. ICI will always ask you to confirm before sharing or deleting important information.
  5. Look for warnings. The system will warn you if you are about to share something outside your private scope.
Warning: Never share passwords, financial info, or highly sensitive data in shared memory.

Collaborating with Others

  • Invite trusted people to your shared memory using the invite or QR code feature.
  • Everyone in a shared memory group can add, edit, or view information (unless you set restrictions).
  • Use chat to discuss, clarify, or update shared memories together.
  • You can always see who has access and remove people at any time.
Tip: Use shared memory for group tasks, family reminders, or collaborative projects. Keep personal notes in your private memory.

Respecting Privacy

  • ICI never shares your private memory unless you explicitly allow it.
  • All sharing actions require your confirmation, often with extra explanation for clarity.
  • Review the Policies page for more on privacy and data handling.

Accessibility & Safety

  • ICI uses a "naggy" confirmation system: it will ask you in multiple ways before taking important actions.
  • All confirmations are accessible for screen readers and assistive tech.
  • If you are unsure, you can always cancel or ask for more information before proceeding.

Need Help?

Check the Changelog for new features, or Policies for privacy details. For more help, contact your administrator or support team.

How to Start the App

Use python app.py to start the ICI Chat backend for all environments.

Note: run_refactored.py is deprecated and should not be used.

QR Code Authentication (May 2025 Update)

  • The chat page displays a QR code above the Ask AI textarea for easy device linking.
  • Once authenticated from another device, the QR code disappears everywhere for that client session.
  • This is synchronized in real time across all open tabs and devices.
  • The join page always shows the QR code for sharing your unique client ID.
